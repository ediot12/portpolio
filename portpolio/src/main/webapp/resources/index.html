- 프로젝트 개요

PowerPack은 Symantec 솔루션을 엔진으로 활용하여 Symantec 보안솔루션의 기능을 더욱더 극대화하여 활용할 수 있도록 제공하고 보안업무 프로세스를 세워주는 역할을 합니다.
주요 개인정보 검출 솔루션인 DLP와 USB, CD와 같은 이동식 매체제어 솔루션인 SEP, 출력물 로깅을 하는 PrintChaser를 승인시스템을 묶어 단말 통합 모니터링을 하는 솔루션 입니다.
여러 외산 제품을 사용하다보면 관리포인트가 늘어나고 사용자마다 또는 제품마다 별도 설정을 해야하는 번거로움이 있어 이를 관리포인트를 줄여주는 것이 PowerPack 제품 입니다.
위 이미지에서 "파워팩 서버" 를 담당하여 추가 기능 개발 & 유지 보수를 담당하였습니다.

- 프로젝트 개요

무료 온라인 수학교육 서비스인 "진단수학" 이라는 앱을 런칭하기 위해 수학 콘텐츠를 사전 관리가 필요합니다.
각 학년별 수학 커리큘럼을 정리하고 , 수학 지식 객체 관리, 수학 문제집 편집 및 관리를 하기 위한 프로젝트가 바로 StemWeb 입니다.
STEM은 Science, Technology, Engineering, Mathematics의 약자이며 과학적으로 분석하여 문제해결력을 높히기 위해 수학 지식 네트워크를 구축해 수학 학습에 적용합니다.
본 프로젝트는 수학교육과/통계학 출신의 연구원들이 콘텐츠 관리를 위해 사용합니다.

- 담당업무

* 수학 지식 객체를 관리하는 화면입니다.
* 지식 객체를 등록하고 상위, 하위 지식 객체를 매핑하여 각 객체간 관계도를 형성합니다.
* 목록 출력에 사용하는 테이블은 DataTables 라이브러리 사용했습니다.

- 담당업무

* 지식객체 등록을 하는 화면입니다.
* 객체 등록은 일반 평문과 수식 입력을 위한 Latex가 있습니다.
* Latex 수식은 사이트 내에 코드를 입력할 시 화면에는 수식으로 변경됩니다.
  Ex) Latex 수식인 \[x=\frac{-b\pm\sqrt{b^2 -4ac}}{2a}\] 를 입력할 시 근의 공식이 출력됩니다.

* 수식 입력은 CkEditor 라이브러리를 사용하였으며 추가 Component인 MathJax 라이브러리를 사용했습니다.

- 담당업무

* 수학 지식 객체를 생성하였다면 객체간 전체 관계도를 보여주는 화면입니다.
* 객체간 상관 관계 표현을 위한 VisJs 라이브러리 사용했습니다.
* 노드간 수치화된 관계 및 노드 위치를 별도로 저장할 수 있는 기능 구현했습니다.
* 노트간 위치는 서버내의 Json 데이터로 별도 저장하여 Read/Write 했으며 노드간 행위는 제공된 문서를 참조하여 개발했습니다.

- 담당업무

* 교과과정 중 특정 학습단원 (최대공약수와 최소공배수) 카테고리에 지식객체를 매핑 후 지식 수준을 보정하는 화면입니다.
* 마르코프 지식네트워크 모델 기반으로 개발된 모듈을 실행하면 평가데이터를 보정할 수 있다.
* 지식 객체를 보정하는 작업은 수학/통계 출신 연구원들이 사용하게 된다.
* 객체간 상관 관계 표현을 위한 VisJs 라이브러리 사용했습니다.
* 평가 데이터를 차트화 시키기 위해 AmCharts 라이브러리 사용했습니다.

- 담당업무

* 학습단원에 들어있는 문제를 관리하는 화면입니다.
* 특정 업체로 부터 제공 받은 문제를 교과과정-학습단원에 맞게 매핑하는 화면입니다.
* 학습단원 트리를 구현하기 위해 JsTree 라이브러리를 사용했습니다.

- 담당업무

* 앞전에 있는 "문제은행" 페이지에서 문제 번호를 클릭하면 출력되는 화면 입니다.
* 문제 - 정답 - 해설로 매핑된 데이터들을 화면에 출력시킵니다.
* 학습단원에 있는 문제, 정답, 해설을 확인하는 기능 정도만 있습니다.

- 담당업무

* 앞전에 있는 "문제은행" 에서 수정 버튼을 클릭시 출력되는 화면 입니다.
* 난이도 / 문제유형 / 상태 / 의견을 작성하여 문제의 기본적인 상태를 입력 할 수 있습니다.
* 좌측은 문제 및 정답을 편집 할 수 있는 화면이고 우측은 모바일 미리보기 화면 입니다.
* 문제 및 정답을 편집 할 수 있게 CkEditor를 사용하였고 수식 표현을 위해 MathJax 라이브러리를 사욯했습니다.
* 문제 및 정답을 편집하면서 모바일 앱에 어떻게 표현이 되는지 확인을 하고 작업을 하기 위함이 주 목적입니다.
